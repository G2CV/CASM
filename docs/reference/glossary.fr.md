# Glossaire

Définitions rapides des termes utilisés dans CASM.

- **Scope**: périmètre autorisé (domaines/IPs/ports/protocoles)
- **ScopeGuard**: mécanisme d'application de la politique avant exécution
- **Evidence**: événement brut en JSONL (source de vérité)
- **Finding**: constat de sécurité dérivé des outils
- **SARIF**: format standardisé pour les intégrations CI/sécurité
- **Baseline**: exécution de référence pour comparaison
- **Diff**: comparaison entre deux exécutions (`added`/`removed`/`unchanged`)
- **Dry-run**: validation de la configuration/politique sans actions réseau
